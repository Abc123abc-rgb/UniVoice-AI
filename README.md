# UniVoice AI  
**One Platform. Every Ability.**

## Overview
UniVoice AI is an AI-powered accessibility and communication platform designed to assist individuals with different abilities, including non-verbal users, blind or low-vision users, deaf or hard-of-hearing users, and elderly individuals.

The platform enables inclusive communication by converting gestures, speech, text, and visual inputs into accessible outputs using artificial intelligence.

## Key Capabilities
- Gesture / Sign to Speech
- Text to Speech with AI-assisted typing
- Speech to Text for deaf users
- Object Detection and Text Recognition for blind users
- Quick Communication Phrases
- Emergency SOS with location sharing
- Multilingual support (English, Hindi, Marathi)
- Customizable voice types (male, female, child, elderly)

## Technology Stack
- Google Gemini API
- Google AI Studio
- TensorFlow
- Google ML Kit (planned integration)

## Use Cases
- Communication support for non-verbal individuals
- Assistance for visually impaired users
- Real-time transcription for hearing-impaired users
- Safety and emergency assistance for elderly users

## Project Status
This project is currently under development.  
The repository is intended for documentation, design references, and hackathon submission purposes. Full implementation will be added after testing and validation.

## Vision
UniVoice AI aims to make communication universal, inclusive, and accessible to everyone using intelligent and human-centered AI solutions.
